Short Description: A simple plugin to manage WordPress user data and options with an easy-to-use interface.

== Description ==

The **User and Options Manager** plugin allows you to manage WordPress user data and options directly from the admin panel with an intuitive interface. This plugin enables administrators to easily update user information and WordPress options like site settings and email addresses.

With this plugin, you can:
- View and manage user data (name, email, etc.).
- Edit user details directly from the admin panel.
- View and manage key WordPress options like site URL, blog name, and admin email.
- Update WP options with just a few clicks.
- Easy interface with a tabbed layout for both user data and WP options.

== Installation ==

1. Upload the `user-options-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the plugin's menu page under **Manage Database** in the WordPress dashboard.
4. Use the interface to manage user data and WP options.
5. Edit user information or update WP options as needed.

== Frequently Asked Questions ==

= How do I manage user data? =

After activating the plugin, go to the **Manage Database** section in the admin menu. You will find a list of users with options to edit their details directly.

= How do I update WP options? =

In the **WP Options** tab, you can view and edit various WordPress options such as the site URL, blog name, and admin email.

= Will this plugin allow me to manage all user data? =

The plugin allows you to view and edit basic user information like name and email. However, more complex user fields (e.g., passwords) are not editable.

= Is it possible to update multiple options at once? =

Currently, the plugin allows one option update at a time. Simply select an option and update it directly from the admin panel.
